# 京东猪脸识别比赛

link: http://jddjr.jd.com/item/4

### 该项目共分 3 个流程

- video_process

    将视频处理成图片数据

    data argument, 对所有图片进行旋转、平移、调亮度、调色度...等等，丰富训练集

- fcn

    处理图片数据，将猪单独切割处理

    data argument，将切割后的猪的图片进行旋转、平移、调亮度、调色度...等等，丰富训练集

- classify

    将每只猪识别出来

    根据 fcn 切割后的猪作为输入，进行识别
